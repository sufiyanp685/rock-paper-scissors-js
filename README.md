# ✊✋✌️ Rock Paper Scissors Game (JavaScript)

A fun and interactive Rock Paper Scissors game built using HTML, CSS, and JavaScript. Play against the computer with real-time results and score tracking.

---

## 🚀 Features

- 🤖 Play against computer
- 🧠 Random computer choice
- 🏆 Winner determination logic
- 📊 Score tracking system
- 🔁 Restart game option
- ⚡ Fast and responsive UI

---

## 🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript (Vanilla JS)

---

- Game logic implementation
- DOM manipulation
